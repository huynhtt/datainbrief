# Dataset on Composite Efficiency Indices for ASEAN FX Markets

## Repository Contents

This repository contains:

- Processed monthly efficiency sub-indices  
- Monthly Composite Efficiency Index (CEI)  
- R scripts reproducing the full data-generation workflow  
- Documentation and metadata  

## Raw Data Policy

Raw daily exchange rate data are **not included** in this repository.

The original exchange rate observations were obtained from publicly accessible financial data providers and official central bank statistical portals. These providers retain ownership of the raw data, and their respective terms of use may restrict redistribution.

To ensure compliance with third-party data policies, this repository distributes only:

- Derived efficiency indicators  
- Aggregated composite indices  
- Reproducible transformation scripts  

Users wishing to fully reproduce the dataset must independently download the original daily exchange rate series from the respective public sources and execute the provided R scripts.
